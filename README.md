# Text2Speech 
Python Script To Convert Text To MP3 
 
Python 3.9.6  
 
Use requirements.txt file to install all required libraries 
 
:ie 
 
   python -m venv venv 

   source venv/bin/activate 

   pip install --upgrade pip 

   pip install -r requirements.txt  

   python TextToSpeech.py
